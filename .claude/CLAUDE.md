# Claude Rules für Status Monitor Frontend

## Wichtige Einschränkungen

- **Niemals `pnpm dev`, `pnpm build`, `pnpm start` oder ähnliche Befehle ausführen**, es sei denn der User fragt explizit danach
- Die Webapp läuft im Hot-Reload, manuelle Starts sind nicht erforderlich

## Projekt-Kontext

- Next.js 16 mit App Router
- Tailwind CSS v4 + shadcn/ui
- React Query für Server-State
- Recharts für Visualisierungen
- TypeScript strict mode
